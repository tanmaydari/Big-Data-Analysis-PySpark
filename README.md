## Project Overview
This project performs Big Data analysis using PySpark on Google Colab. The Netflix dataset was processed to extract insights and demonstrate scalable data processing.

## Tech Stack
- Python
- PySpark
- Google Colab

## Features
- Data loading and cleaning
- Year-wise content analysis
- Country-wise distribution
- Content type analysis (Movies vs TV Shows)
- Rating analysis
- Demonstration of scalability using partitioning and caching

## Insights
- Movies dominate Netflix content
- USA produces the highest content
- Content growth increased after 2015
- TV-MA is the most common rating

## Scalability
The dataset was processed using distributed computing features of PySpark including:
- Repartitioning
- In-memory caching
